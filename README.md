# devion - Client

## Projeto iniciado com Vue CLI - @vue/cli 4.5.11

- Vue 2.x
- VueX
- Vue Router
- ESLint
- Prettier
- Lodash
- Vuetify
- Vuelidate
- Moment

## Para Instalar as dependências do projeto

```
npm install
```

### Compila e ativa o hot-reload para desenvolvimento

```
npm run serve
```

### Compila e minifica para produção

```
npm run build
```

### Rodar testes unitários

```
npm run test:unit
```

### Lint

```
npm run lint
```
